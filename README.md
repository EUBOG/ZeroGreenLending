# EcoLife — Лендинг экотоваров для дома

Одностраничный лендинг для стартапа по продаже экологичных товаров. Создан с использованием современных подходов вёрстки (HTML + CSS + JS) и вайб-кодинга для плавных анимаций.

## Структура

```
index.html       — основной файл (структура + SEO-мета)
css/style.css    — все стили (фирменные цвета, анимации, адаптив)
js/script.js     — 11 интерактивных функций
```

## Секции лендинга

| # | Блок | ID | Описание |
|---|---|---|---|
| 1 | Первый экран | `#hero` | Заголовок, подзаголовок, CTA-кнопки, статистика |
| 2 | О продукте | `#about` | Описание EcoLife, сертификаты, визуальная сетка |
| 3 | Преимущества | `#benefits` | 4 карточки: материалы, Zero Waste, выгода, доставка |
| 4 | Для кого | `#forwhom` | 4 сегмента ЦА: семьи, миллениалы, подарки, B2B |
| 5 | Статистика | — | Анимированные счётчики (5000+ семей, 12000+ деревьев) |
| 6 | Товары | `#products` | 4 карточки товаров с ценами и бейджами |
| 7 | Кейс | `#case` | Реальный пример использования (семья Ивановых) |
| 8 | Отзывы | `#testimonials` | 3 отзыва с рейтингом 5★ |
| 9 | Форма заявки | `#cta` | Скидка 15% + бесплатная доставка |
| 10 | Контакты | `#contact` | Форма обратной связи |

## Анимации (вайб-кодинг)

1. **Gradient Shift** — живой градиент фона hero
2. **Float Leaf** — 6 парящих листьев по всей странице
3. **FadeInUp** — появление блоков при скролле (IntersectionObserver)
4. **Ripple** — волна на кнопках при клике
5. **ModalIn** — анимация модального окна
6. **Parallax** — движение элементов hero за курсором
7. **Counter** — анимированные счётчики (ease-out cubic)
8. **Scroll Bar** — индикатор прогресса чтения
9. **Header Shrink** — сжатие шапки при скролле

## Интерактив

- Адаптивное мобильное меню
- Ripple-эффект на всех кнопках
- Форма заявки с уведомлением (без перезагрузки)
- Модалка с промокодом ECOLIFE15

## Запуск

Открой `index.html` в браузере — всё работает из коробки.
